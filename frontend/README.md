# Frontend (React + TypeScript + Vite)

## Available Scripts

- `npm run dev` - Start local development server.
- `npm run build` - Run typecheck and production build.
- `npm run lint` - Lint frontend source files.
- `npm run preview` - Preview production build.

## Notes

- Main app entry: `src/main.tsx`
- Main page component: `src/App.tsx`
- Global styles: `src/index.css`

This frontend has been cleaned to a minimal starter state so feature development can begin immediately.
